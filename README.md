# 🎧 Spotify Data Analytics Project

## 📌 Overview

This project focuses on analyzing Spotify music data to uncover insights about song features, music trends, user preferences, and factors affecting song popularity.

---

## 🎯 Objectives

* Analyze music trends over time
* Identify popular genres and artists
* Study song features like danceability, energy, and tempo
* Understand relationships between different features

---

## 📂 Datasets Used

* `data.csv` → Main dataset (song features)
* `data_by_genres.csv` → Genre-wise data
* `data_by_year.csv` → Year-wise trends
* `data_by_artist.csv` → Artist-wise data
* `data_w_genres.csv` → Combined dataset

---

## 🛠️ Tools & Technologies

* Python
* Pandas, NumPy
* Matplotlib, Seaborn


---

## 🔍 Key Analysis Performed

* Data cleaning 
* Exploratory Data Analysis (EDA)
* Feature distribution analysis
* Trend analysis over years
* Correlation analysis (heatmap)
* Visualization (bar charts, line plots, scatter plots)

---

## 📊 Key Insights

* Most songs have moderate energy and danceability
* Music popularity changes significantly over time
* Certain genres dominate listener preferences
* Energy and danceability show positive correlation
* Popular artists strongly influence listening trends
